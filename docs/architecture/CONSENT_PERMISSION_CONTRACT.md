# Consent And Permission Contract

Sensitive user data may be shared only when all checks pass:

- User consent scope allows the share.
- Tenant role permission allows the access.
- Sensitivity level allows the target audience.
- Report redaction policy has been applied.
- Audit log is written.

Sensitive modules:

- crisis
- confession
- pastoral care
- healing journey
- private prayer
- memory/profile
- reports

Default stance:

- Individual private data remains private.
- Mentor, group, pastor, and care-team access is permission-scoped.
- Organization admins receive operational metadata, not private spiritual content, unless explicit consent and role permission both apply.
